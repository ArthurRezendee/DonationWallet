About

Hello my name is Arthur Rezende i'm a web3 student and i created this project to learn more about smart contracts. For this I created a smart contract on the Sepolia testnet of Ethereum using solidity. The contract receives a value and stores it at its address, only the owner can withdraw this money. Information such as donors and amount of donations are stored and displayed here! To display such information I made a simple rest api using node.js that is integrated with etherscan and ether.js!
